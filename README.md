# factorio-recipe-vis
A visualization of recipes in Factorio using D3.js



## Libraries Used

* d3 v3.5.17
* d3-sankey v0.7.1
* d3-chart v0.3.0
* [q-m/d3-chart-sankey](https://creativecommons.org/licenses/by-nc-sa/3.0/)
* JQuery v3.3.1
* [kevinta893/factorio-recipes-json](https://github.com/kevinta893/factorio-recipes-json)
